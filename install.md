# Installation Requirements

## Required software

- Java JDK 21
- Apache Maven
- Node.js 20 or 22 LTS
- npm
- Git

## Direct download links

### Java JDK 21
- Oracle JDK 21: https://www.oracle.com/java/technologies/downloads/
- OpenJDK 21: https://jdk.java.net/21/

### Apache Maven
- https://maven.apache.org/download.cgi

### Node.js
- https://nodejs.org/

### Git
- https://git-scm.com/downloads

## Required versions

- Java: 21
- Maven: latest stable
- Node.js: 20.x or 22.x LTS
- npm: comes with Node.js

## Quick check
Run these commands after installing:

```bash
java -version
mvn -version
node -v
npm -v
```

## Notes
- Backend uses Java + Maven
- Frontend uses Node.js + npm
- No separate database installation is required for basic run because the backend uses H2 by default
